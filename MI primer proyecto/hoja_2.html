01 · CONCEPTO BASE

¿Qué es una guía de estilo?

Conjunto de reglas y convenciones para escribir código de forma uniforme dentro de un proyecto o equipo.

“Es mucho más fácil entender una base de código grande cuando todo el código mantiene un estilo consistente”.

Fuente: Google Style Guides [1]

ORDEN
Reduce decisiones repetidas sobre formato y nombres.
LECTURA
Hace que el código sea predecible para otros programadores.
EQUIPO
Facilita revisiones, mantenimiento y colaboración.
PALABRAS CLAVE: reglas · convenciones · consistencia · legibilidad
