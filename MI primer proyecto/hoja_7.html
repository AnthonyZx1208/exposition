Meta/Facebook y convenciones en React

React organiza la interfaz en componentes reutilizables. Su documentación indica que los componentes son funciones de JavaScript cuyos nombres deben iniciar con letra mayúscula.

Principio clave

Si el nombre empieza con mayúscula, React lo interpreta como componente; si está en minúscula, lo trata como etiqueta HTML.

function ProfileCard() {
return <ProfileCard userName="Ana" />;
}

Fuente: React Docs, “Your First Component” [4]

PascalCase
Para componentes: ProfileCard, UserMenu.
camelCase
Para props y variables: userName, isActive.
JSX
Permite escribir marcado dentro de JavaScript.
Modularidad
Separar componentes en archivos cuando el proyecto crece.
